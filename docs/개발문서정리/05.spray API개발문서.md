## ✅ 1. API 개발 및 수정 사항

### Spray API 개발
- **엔드포인트**: `/api/v1/spray`
- **기능**: 채팅방에서 금액을 뿌리는(spray) 기능 구현
- **주요 구현 사항**:
  - 3자리 랜덤 토큰 생성
  - 금액 분배 로직
  - 뿌리기 생성 API

#### 예제 요청/응답

```json
// POST /api/v1/spray
Request:
{
    "total_amount": 5000,
    "recipient_count": 3
}
Headers:
{
    "X-USER-ID": "1",
    "X-ROOM-ID": "TEST-ROOM-1"
}

Response:
{
    "token": "a1b"  // 3자리 랜덤 토큰
}
```

## ✅ 2. 버그 수정 사항

### 토큰 변수 미정의 오류
- **문제**: 테스트 코드에서 "token"이 정의되지 않은 오류 발생
- **원인**: 변수명 불일치
- **해결**: 변수명을 `token`에서 `spray_token`으로 통일

```python
# 수정 전
token = await service.create_spray(...)
assert len(token) == 3

# 수정 후
spray_token = await service.create_spray(...)
assert len(spray_token) == 3
```

def generate_token() -> str:

def distribute_amount(total: int, count: int) -> List[int]:
    # 총액을 인원수만큼 랜덤하게 분배
    if total < count:
        raise ValueError("Total amount must be greater than recipient count")

## ✅ 4. 주요 이슈 해결

### 금액 분배 로직 구현
- **문제**: 총액을 인원수만큼 공평하게 분배하는 로직 필요
- **원인 분석**: 
  - 모든 사용자가 최소 1원 이상 받아야 함
  - 총액이 인원수보다 작으면 안됨
  - 랜덤하게 분배되어야 함
- **해결 방법**: 
  1. 남은 금액에서 랜덤하게 금액 할당
  2. 마지막 사람에게 남은 금액 전체 할당
  3. 모든 금액이 1원 이상이 되도록 보장

```python
def distribute_amount(total: int, count: int) -> List[int]:
    if total < count:
        raise ValueError("Total amount must be greater than recipient count")
    
    amounts = []
    remaining = total
    
    for i in range(count - 1):
        # 남은 사람들에게 최소 1원씩은 줄 수 있도록 계산
        max_amount = remaining - (count - i - 1)
        amount = random.randint(1, max_amount)
        amounts.append(amount)
        remaining -= amount
    
    # 마지막 남은 금액 추가
    amounts.append(remaining)
    return amounts
```
